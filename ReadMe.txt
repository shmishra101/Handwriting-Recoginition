HandWriting Recoginition 

This project implements the linear regression,logistic regression and neural network on Human Observed and GSC dataset to recoginize the "and" written by multiple pairs.
DataSet has been provided in Introduction to Machine Learning class at SUNY Buffalo.
A dataset has been provided that consist of multiple  "and"  images written by several writers.Two dataset consist of features extracted from the "and" written images.
First of all the features are mapped corresponding to their writers.
All three models learn this features and predict the "and" belong to which writer.
